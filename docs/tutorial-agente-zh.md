## 1. 系统介绍

**Argencargo** 是一家阿根廷货运代理公司，专门从事中国和美国到阿根廷的进口业务。您作为**中国代理**，负责处理 **航空 A（商业快递）** 渠道的货物。

### 您的主要职责

- 📦 在中国仓库**接收客户的包裹**
- 📝 在系统中**注册每个包裹**（物流单号、尺寸、重量）
- ✈️ 等待管理员创建**航班**，将多个客户的包裹合并发送
- 📤 **发送货物**到阿根廷，并填写国际物流单号
- 💰 管理您的**往来账户**（余额、预付款、扣除）

### 系统语言

界面支持**中文**和**西班牙语**。在登录页面或主页面右上角可以切换语言。**🇨🇳 中文 / 🇦🇷 ES**

## 2. 注册和登录

### 系统访问地址

打开浏览器，访问：

**👉 https://argencargo.com.ar/agente**

### 首次注册

1. 在登录页面，点击 **"注册"** （或选择 _"没有账户？注册"_）
2. 填写以下信息：
   - **名字** （必填）
   - **姓氏**
   - **电子邮件** （必填，请使用您日常使用的邮箱）
   - **密码** （至少 6 个字符）
3. 点击 **"注册"** 按钮
4. 系统会显示：**"账户待批准"**

### 登录现有账户

1. 输入您的**电子邮件**和**密码**
2. 点击 **"登录"**
3. 进入主面板

## 3. 等待批准

注册后，您的账户处于**待批准状态**。这意味着：

- ⏳ 管理员需要手动批准您的账户
- 📧 批准后您会收到通知（通过电子邮件或微信）
- ✅ 批准后，您可以登录并开始使用系统

### 批准期间您会看到

```
账户待批准
您的注册已发送给管理员。批准后我们会通知您。
```

如果超过 24 小时仍未批准，请联系管理员。

## 5. 仓库：注册包裹

这是您**最常用**的功能。每当客户的包裹到达您的仓库时，您必须立即注册它。

### 5.1 注册新包裹的步骤

1. 进入 **"仓库"** 标签
2. 点击右上角的 **"+ 注册包裹"** 按钮
3. 填写表单（详见下方）

### 5.2 选择客户

每个包裹必须关联一个客户。您可以通过两种方式找到客户：

**方法 A：客户代码**
- 客户会告诉您他们的代码（例如 `JUAPED`、`FRASUA`、`GUSGAL`）
- 在 **"客户代码"** 输入框中输入
- 系统会自动验证：✅ "客户已找到" 或 ❌ "未找到代码"

**方法 B：搜索客户**
- 点击 **"搜索"** 按钮
- 按代码或姓名搜索（例如输入 "Juan" 或 "JUAPED"）
- 从列表中选择正确的客户

### 5.3 客户没有注册账户？

如果客户告诉您他们没有 Argencargo 账户：

1. 点击 **"未注册的客户"** 选项
2. 系统会显示提示：
   > _"包裹将保持未分配状态。Argencargo稍后将分配它。"_
3. 您仍然可以注册包裹，但它会进入"无主"列表
4. 管理员稍后会将包裹分配给正确的客户

### 5.4 客户已有开放的操作

如果该客户已经有进行中的操作（例如他们之前已经发送了几个包裹）：

- 系统会显示：**"该客户有开放的操作。此包裹将添加到：[操作号 AC-XXXX]"**
- 您的新包裹会**自动添加到该操作**（合并发货 / 整合）
- 客户最终会一起收到所有包裹

如果客户**没有**开放的操作：

- 系统会显示：**"将为该客户创建新操作"**
- 系统会自动创建新的操作号（例如 AC-0099）

### 5.5 包裹信息

填写以下字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| **物流单号** (Tracking) | 中国国内物流的单号 | `ECZEN31902219401` |
| **重量** (kg) | 实际称重 | `2.5` |
| **长** (cm) | 包裹的长度 | `45` |
| **宽** (cm) | 包裹的宽度 | `30` |
| **高** (cm) | 包裹的高度 | `35` |

### 5.6 多个包裹（同一客户）

如果同一客户同时寄来多个包裹：

1. 填写第一个包裹的信息
2. 点击 **"+ 添加另一个包裹"**
3. 系统会显示 _"包裹 1"_、_"包裹 2"_、_"包裹 3"_…
4. 为每个包裹填写独立的信息（每个包裹都需要自己的物流单号、尺寸、重量）

### 5.7 保存包裹

填写完所有信息后：

1. 点击 **"保存包裹"** 按钮
2. 系统显示：✅ **"包裹注册成功"**
3. 包裹现在出现在 **"已注册的包裹"** 列表中

### 5.8 查看已注册的包裹

在 **仓库** 标签的下方，您可以看到所有已注册的包裹：

| 列 | 说明 |
|------|------|
| 操作 | 操作号（AC-XXXX） |
| 客户 | 客户代码 + 名字 |
| 物流单号 | 您填写的中国物流单号 |
| 重量 | kg |
| 日期 | 注册日期 |

可以使用顶部的**搜索框**按客户代码、姓名或物流单号过滤。

## 6. 活跃航班：发送货物

包裹在仓库累积后，**管理员**会决定何时创建航班来发送它们。

### 6.1 工作流程

1. ✅ 您注册包裹 → 包裹进入"仓库"
2. ✅ 管理员从仓库选择多个包裹（可能来自多个客户）
3. ✅ 管理员创建**新航班**并分配给您
4. ⏳ 您在 **"活跃航班"** 标签中看到该航班
5. ⏳ 您等待管理员上传**出口发票**
6. ✅ 管理员上传发票 → 状态变为"可以发送"
7. ✅ 您打包并发送货物到阿根廷
8. ✅ 您在系统中填写**国际物流单号**和发送信息

### 6.2 查看航班

在 **"活跃航班"** 标签中：

- 列出所有分配给您的进行中航班
- 每个航班显示：
  - **航班号**
  - **状态**：准备中 / 可以发送 / 已发送 / 已接收
  - **包含的操作数**

点击航班可以查看详情。

### 6.3 航班状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| **准备中** | 🟡 黄色 | 管理员正在准备航班，等待发票 |
| **可以发送** ✓ | 🟢 绿色 | 发票已上传，您可以发货 |
| **已发送** | 🔵 蓝色 | 货物已离开仓库，正在运输中 |
| **已接收** | 🟢 绿色 | 货物已到达阿根廷 |

### 6.4 进入航班详情

点击航班行 → 进入详情页面，您会看到：

#### 上半部分：航班信息

- **航班号**
- **状态**
- **航班操作**：包含的所有操作（AC-XXXX）和它们的包裹
- **目的地地址**：阿根廷收货地址
- **出口发票**：管理员上传的 PDF（点击 **"下载发票"**）

#### 下半部分：发送表单（仅"可以发送"状态）

当航班状态为 **"🚀 可以发送"** 时，系统会显示发货表单：

| 字段 | 说明 |
|------|------|
| **总重量 (kg)** | 自动从包裹计算，您可以编辑实际重量 |
| **总费用 (USD)** | 您和阿根廷之间约定的运费 |
| **国际物流单号** | DHL/UPS/FedEx 等的单号 |
| **快递** | 选择运输公司：DHL、UPS、FedEx 等 |
| **付款方式** | - **往来账户**：从您的余额扣除<br>- **现金**：现金支付<br>- **转账**：银行转账 |

### 6.5 确认发送

填写完所有字段后：

1. 检查信息**准确性**
2. 点击 **"确认发送"** 按钮
3. 系统会显示警告：
   > _"确认后，您将无法编辑航班。操作将更改为'运输中'。"_
4. 再次确认 → 航班状态变为 **"已发送"**
5. 包裹的所有客户**自动**收到通知（通过 WhatsApp + 邮件）

⚠️ **重要**：确认发送后，您**无法**修改任何信息。请仔细检查。

## 8. 我的账户：余额和预付款

进入 **"我的账户"** 标签查看您的财务信息。

### 8.1 余额

页面顶部显示当前**余额（USD）**：

- 🟢 **正余额**：Argencargo 给您的预付款多于您发送的航班费用 → 您可以用它支付未来的航班
- 🔴 **负余额**：您发送的航班费用多于预付款 → 您需要补充资金或 Argencargo 会调整

### 8.2 历史动向

显示所有动向：

| 类型 | 说明 |
|------|------|
| **预付款** (Anticipo) | Argencargo 给您的资金（增加余额） |
| **扣除** (Deducción) | 用 "往来账户" 方式发送的航班费用（减少余额） |

每个动向显示：
- 日期
- 类型
- 金额
- 描述（例如 _"航班 VL-2026-04 的费用"_）

### 8.3 当余额低时怎么办？

1. 联系管理员请求新的预付款
2. 或者在发送下一个航班时使用**现金**或**转账**而不是**往来账户**

## 10. 典型工作日流程

### 早上：检查仓库

1. 登录系统 → **仓库** 标签
2. 检查昨天是否有未注册的新包裹（如果是助手收的）
3. 处理任何待办事项

### 白天：注册包裹

每当客户的包裹到达：

1. 接收包裹
2. **立即称重和测量**（不要积累，避免遗忘）
3. 登录系统 → **+ 注册包裹**
4. 填写：客户代码 + 物流单号 + 尺寸 + 重量
5. 保存

### 等待管理员：

- 不需要您做任何事
- 管理员会决定何时合并包裹并创建航班
- 您会在 **活跃航班** 中看到分配的新航班

### 准备发送（当航班状态变为"可以发送"时）：

1. 进入 **活跃航班** → 点击航班
2. 下载**出口发票**
3. 打印发票，准备包裹
4. 联系您的国际快递（DHL/UPS/FedEx）
5. 提供发票和包裹
6. 获得**国际物流单号**
7. 回到系统：填写发送表单
8. 点击 **"确认发送"**

### 一周/一个月：

- 在 **统计** 标签检查您的总数据
- 在 **我的账户** 检查余额，必要时请求预付款

## 11. 重要规则和注意事项

### ⚠️ 一定要做

✅ **每个包裹必须有物流单号** — 没有单号无法追踪货物
✅ **正确称重和测量** — 系统会按"可计费重量"（实际重量 vs. 体积重量）计算运费
✅ **确认客户代码** — 错误的代码会导致包裹分配给错误的人
✅ **每天注册** — 不要让包裹积累不注册
✅ **发送前检查地址** — 错误的地址会导致丢失货物

### ❌ 不要做

❌ **不要不询问就修改包裹信息** — 如果有疑问，联系管理员
❌ **不要在没有发票的情况下发送** — 等待"可以发送"状态
❌ **不要假设管理员看到了所有内容** — 重要事项请通过 WhatsApp 通知
❌ **不要分享您的密码** — 每个代理都有自己的账户

### 📐 测量和重量提示

- **实际重量**：用秤称量
- **体积重量**：长 × 宽 × 高 / 5000（cm）= 体积重量（kg）
- **可计费重量**：取两者之间较大的值
- 系统会自动计算 — 您只需输入实际重量和尺寸

例如：
- 包裹尺寸 50×40×30 cm，实际重量 5 kg
- 体积重量 = 50×40×30/5000 = 12 kg
- 可计费重量 = max(5, 12) = **12 kg**

## 12. 常见问题 FAQ

### Q1: 我注册了账户，但还没有被批准。怎么办？

**A**: 通常 24 小时内批准。如果超过 24 小时，请通过 WhatsApp 联系管理员。

### Q2: 我输错了客户代码，怎么修改？

**A**: 已注册的包裹**不能直接修改**客户。请联系管理员，他可以重新分配。

### Q3: 客户没有 Argencargo 账户，怎么办？

**A**: 选择 **"未注册的客户"** 选项注册包裹。包裹会暂时无主，管理员稍后会分配。

### Q4: 我看到一个航班但没有发票。我能发送吗？

**A**: **不能**。等待状态变为 **"可以发送"** ✓。如果几天后还没有发票，请通过 WhatsApp 询问管理员。

### Q5: 国际快递给了我跟踪号，我应该用 DHL 还是其他？

**A**: 使用客户实际使用的快递公司。在 **"快递"** 字段中选择 DHL、UPS、FedEq 或其他。这影响系统跟踪货物的方式。

### Q6: 我可以编辑已确认的航班吗？

**A**: **不能**。一旦点击 **"确认发送"**，航班锁定。请仔细检查后再确认。

### Q7: 余额变成负数了。怎么办？

**A**: 联系管理员请求新的预付款，或者下一个航班使用现金/转账而不是往来账户。

### Q8: 我无法登录，密码忘了怎么办？

**A**: 联系管理员重置密码。

### Q9: 我可以同时为多个航班工作吗？

**A**: 可以。您可以同时有多个进行中的航班。在 **活跃航班** 中可以看到所有航班。

### Q10: 系统是否离线工作？

**A**: 不行，需要网络连接。请确保有稳定的互联网。
